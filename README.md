- PostgreSQL

# Como executar o projeto

## Back end
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone https://github.com/eijilucas/spring-validation

# entrar na pasta do projeto back end
cd spring-validation

# executar o projeto
./mvnw spring-boot:run
```
